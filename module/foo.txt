<h1>3. Modules<a class="headerlink" href="http://anandology.com/python-practice-book/modules.html#modules" title="Permalink to this headline">¶</a></h1>
<p>Modules are reusable libraries of code in Python. Python comes with many standard library modules.</p>
<p>A module is imported using the <cite>import</cite> statement.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">time</span>
<span class="gp">&gt;&gt;&gt; </span><span class="k">print</span> <span class="n">time</span><span class="o">.</span><span class="n">asctime</span><span class="p">()</span>
<span class="go">'Fri Mar 30 12:59:21 2012'</span>
</pre></div>
</div>
<p>In this example, we’ve imported the <cite>time</cite> module and called the <cite>asctime</cite>
function from that module, which returns current time as a string.</p>
<p>There is also another way to use the import statement.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">from</span> <span class="nn">time</span> <span class="kn">import</span> <span class="n">asctime</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">asctime</span><span class="p">()</span>
<span class="go">'Fri Mar 30 13:01:37 2012'</span>
</pre></div>
</div>
<p>Here were imported just the <cite>asctime</cite> function from the <cite>time</cite> module.</p>
<p>The <cite>pydoc</cite> command provides help on any module or a function.</p>
<div class="highlight-text"><div class="highlight"><pre>$ pydoc time
Help on module time:

NAME
    time - This module provides various functions to manipulate time values.
...

$ pydoc time.asctime
Help on built-in function asctime in time:

time.asctime = asctime(...)
    asctime([tuple]) -&gt; string
...
</pre></div>
</div>
<p>On Windows, the <cite>pydoc</cite> command is not available. The work-around is to use, the built-in <cite>help</cite> function.</p>
<div class="highlight-text"><div class="highlight"><pre>&gt;&gt;&gt; help('time')
Help on module time:

NAME
    time - This module provides various functions to manipulate time values.
...
</pre></div>
</div>
<p>Writing our own modules is very simple.</p>
<p>For example, create a file called <cite>num.py</cite> with the following content.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="k">def</span> <span class="nf">square</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>

<span class="k">def</span> <span class="nf">cube</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>
</pre></div>
</div>
<p>Now open Python interterter:</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">num</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">num</span><span class="o">.</span><span class="n">square</span><span class="p">(</span><span class="mi">3</span><span class="p">)</span>
<span class="go">9</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">num</span><span class="o">.</span><span class="n">cube</span><span class="p">(</span><span class="mi">3</span><span class="p">)</span>
<span class="go">27</span>
</pre></div>
</div>
<p>Thats all we’ve written a python library.</p>
<p>Try <cite>pydoc num</cite> (<cite>pydoc.bat numbers</cite> on Windows) to see documentation for this numbers modules. It won’t have any documentation as we haven’t providied anything yet.</p>
<p>In Python, it is possible to associate documentation for each module, function using docstrings. Docstrings are strings written at the top of the module or at the beginning of a function.</p>
<p>Lets try to document our <cite>num</cite> module by changing the contents of <cite>num.py</cite></p>
<div class="highlight-python"><div class="highlight"><pre><span class="sd">"""The num module provides utilties to work on numbers.</span>

<span class="sd">Current it provides square and cube.</span>
<span class="sd">"""</span>

<span class="k">def</span> <span class="nf">square</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="sd">"""Computes square of a number."""</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>

<span class="k">def</span> <span class="nf">cube</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="sd">"""Computes cube of a number."""</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>
</pre></div>
</div>
<p>The pydoc command will now show us the doumentation nicely formatted.</p>
<div class="highlight-text"><div class="highlight"><pre>Help on module num:

NAME
    num - The num module provides utilties to work on numbers.

FILE
    /Users/anand/num.py

DESCRIPTION
    Current it provides square and cube.

FUNCTIONS
    cube(x)
        Computes cube of a number.

    square(x)
        Computes square of a number.
</pre></div>
</div>
<p>Under the hood, python stores the documentation as a special field called <cite>__doc__</cite>.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">os</span>
<span class="gp">&gt;&gt;&gt; </span><span class="k">print</span> <span class="n">os</span><span class="o">.</span><span class="n">getcwd</span><span class="o">.</span><span class="n">__doc__</span>
<span class="go">getcwd() -&gt; path</span>

<span class="go">Return a string representing the current working directory.</span>
</pre></div>
</div>

<div class="highlight-text"><div class="highlight"><pre>$ python extcount.py src/

1 csv
</pre></div>
</div>
<p><strong>Problem 3: </strong>Write a program to list all the files in the given directory along
with their length and last modification time. The output should contain one
line for each file containing filename, length and modification date separated
by tabs.</p><p>Hint: see help for <tt class="docutils literal"><span class="pre">os.stat</span></tt>.</p>
<p></p>
<p><strong>Problem 4: </strong>Write a program to print directory tree. The program should take

a tree.</p>

|-- a.txt
|-- b.txt

<div class="section" id="zipfile-module">
<h3>3.1.5. zipfile module<a class="headerlink" href="http://anandology.com/python-practice-book/modules.html#zipfile-module" title="Permalink to this headline">¶</a></h3>
<p>The <cite>zipfile</cite> module provides interface to read and write zip files.</p>
<p>Here are some examples to demonstate the power of zipfile module.</p>
<p>The following example prints names of all the files in a zip archive.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="kn">import</span> <span class="nn">zipfile</span>
<span class="n">z</span> <span class="o">=</span> <span class="n">zipfile</span><span class="o">.</span><span class="n">ZipFile</span><span class="p">(</span><span class="s">"a.zip"</span><span class="p">)</span>
<span class="k">for</span> <span class="n">name</span> <span class="ow">in</span> <span class="n">z</span><span class="o">.</span><span class="n">namelist</span><span class="p">():</span>
    <span class="k">print</span> <span class="n">name</span>
</pre></div>
</div>
<p>The following example prints each file in the zip archive.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="kn">import</span> <span class="nn">zipfile</span>
<span class="n">z</span> <span class="o">=</span> <span class="n">zipfile</span><span class="o">.</span><span class="n">ZipFile</span><span class="p">(</span><span class="s">"a.zip"</span><span class="p">)</span>
<span class="k">for</span> <span class="n">name</span> <span class="ow">in</span> <span class="n">z</span><span class="o">.</span><span class="n">namelist</span><span class="p">():</span>
    <span class="k">print</span>
    <span class="k">print</span> <span class="s">"FILE:"</span><span class="p">,</span> <span class="n">name</span>
    <span class="k">print</span>
    <span class="k">print</span> <span class="n">z</span><span class="o">.</span><span class="n">read</span><span class="p">(</span><span class="n">name</span><span class="p">)</span>
</pre></div>
</div>
<p><strong>Problem 11: </strong>Write a python program <cite>zip.py</cite> to create a zip file. The program should take name of zip file as first argument and files to add as rest of the arguments.</p>
<div class="highlight-text"><div class="highlight"><pre>$ python zip.py foo.zip file1.txt file2.txt
</pre></div>
</div>
<p><strong>Problem 12: </strong>Write a program <cite>mydoc.py</cite> to implement the functionality of pydoc. The program should take the module name as argument and print documentation for the module and each of the functions defined in that module.</p>
<div class="highlight-text"><div class="highlight"><pre>$ python mydoc.py os
Help on module os:

getcwd()
    ...

</div>

<div class="section" id="installing-third-party-modules">
<h2>3.2. Installing third-party modules<a class="headerlink" href="http://anandology.com/python-practice-book/modules.html#installing-third-party-modules" title="Permalink to this headline">¶</a></h2>
<p>PyPI, The Python Package Index maintains the list of Python packages available. The third-party module developers usually register at PyPI and uploads their packages there.</p>
<p>The standard way to installing a python module is using <cite>pip</cite> or <cite>easy_install</cite>. Pip is more modern and perferred.</p>
<p>Lets start with installing <cite>easy_install</cite>.</p>
<ul class="simple">
<li>Download the easy_install install script <a class="reference external" href="http://peak.telecommunity.com/dist/ez_setup.py">ez_setup.py</a>.</li>
<li>Run it using Python.</li>
</ul>
<p>That will install <tt class="docutils literal"><span class="pre">easy_install</span></tt>, the script used to install third-party python packages.</p>
<p>Before installing new packages, lets understand how to manage virtual environments for installing python packages.</p>
<p>Earlier the only way of installing python packages was system wide. When used this way, packages installed for one project can conflict with other and create trouble. So people invented a way to create isolated Python environment to install packages. This tool is called <a class="reference external" href="http://www.virtualenv.org/">virtualenv</a>.</p>
<p>To install <tt class="docutils literal"><span class="pre">virtualenv</span></tt>:</p>
<blockquote>
<div>$ easy_install virtualenv</div></blockquote>
<p>Installing virtualenv also installs the <cite>pip</cite> command, a better replace for <cite>easy_install</cite>.</p>
<p>Once it is installed, create a new virtual env by running the <tt class="docutils literal"><span class="pre">virtualenv</span></tt> command.</p>
<div class="highlight-text"><div class="highlight"><pre>$ virtualenv testenv
</pre></div>
</div>
<p>Now to switch to that env.</p>
<p>On UNIX/Mac OS X:</p>
<div class="highlight-text"><div class="highlight"><pre>$ source testenv/bin/activate
</pre></div>
</div>
<p>On Windows:</p>
<div class="highlight-text"><div class="highlight"><pre>&gt; testenv\Scripts\activate
</pre></div>
</div>
<p>Now the virtualenv <cite>testenv</cite> is activated.</p>
<p>Now all the packages installed will be limited to this virtualenv. Lets try to install a third-party package.</p>
<div class="highlight-text"><div class="highlight"><pre>$ pip install tablib
</pre></div>
</div>
<p>This installs a third-party library called <tt class="docutils literal"><span class="pre">tablib</span></tt>.</p>
<p>The <tt class="docutils literal"><span class="pre">tablib</span></tt> library is a small little library to work with tabular data and write csv and Excel files.</p>
<p>Here is a simple example.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="c"># create a dataset</span>
<span class="n">data</span> <span class="o">=</span> <span class="n">tablib</span><span class="o">.</span><span class="n">Dataset</span><span class="p">()</span>

<span class="c"># Add rows</span>
<span class="n">data</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"A"</span><span class="p">,</span> <span class="mi">1</span><span class="p">])</span>
<span class="n">data</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"B"</span><span class="p">,</span> <span class="mi">2</span><span class="p">])</span>
<span class="n">data</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"C"</span><span class="p">,</span> <span class="mi">3</span><span class="p">])</span>

<span class="c"># save as csv</span>
<span class="k">with</span> <span class="nb">open</span><span class="p">(</span><span class="s">'test.csv'</span><span class="p">,</span> <span class="s">'wb'</span><span class="p">)</span> <span class="k">as</span> <span class="n">f</span><span class="p">:</span>
    <span class="n">f</span><span class="o">.</span><span class="n">write</span><span class="p">(</span><span class="n">data</span><span class="o">.</span><span class="n">csv</span><span class="p">)</span>

<span class="c"># save as Excel</span>
<span class="k">with</span> <span class="nb">open</span><span class="p">(</span><span class="s">'test.xls'</span><span class="p">,</span> <span class="s">'wb'</span><span class="p">)</span> <span class="k">as</span> <span class="n">f</span><span class="p">:</span>
    <span class="n">f</span><span class="o">.</span><span class="n">write</span><span class="p">(</span><span class="n">data</span><span class="o">.</span><span class="n">xls</span><span class="p">)</span>

<span class="c"># save as Excel 07+</span>
<span class="k">with</span> <span class="nb">open</span><span class="p">(</span><span class="s">'test.xlsx'</span><span class="p">,</span> <span class="s">'wb'</span><span class="p">)</span> <span class="k">as</span> <span class="n">f</span><span class="p">:</span>
    <span class="n">f</span><span class="o">.</span><span class="n">write</span><span class="p">(</span><span class="n">data</span><span class="o">.</span><span class="n">xlsx</span><span class="p">)</span>
</pre></div>
</div>
<p>It is even possible to create multi-sheet excel files.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="n">sheet1</span> <span class="o">=</span> <span class="n">tablib</span><span class="o">.</span><span class="n">Dataset</span><span class="p">()</span>
<span class="n">sheet1</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"A1"</span><span class="p">,</span> <span class="mi">1</span><span class="p">])</span>
<span class="n">sheet1</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"A2"</span><span class="p">,</span> <span class="mi">2</span><span class="p">])</span>

<span class="n">sheet2</span> <span class="o">=</span> <span class="n">tablib</span><span class="o">.</span><span class="n">Dataset</span><span class="p">()</span>
<span class="n">sheet2</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"B1"</span><span class="p">,</span> <span class="mi">1</span><span class="p">])</span>
<span class="n">sheet2</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"B2"</span><span class="p">,</span> <span class="mi">2</span><span class="p">])</span>


<span class="n">book</span> <span class="o">=</span> <span class="n">tablib</span><span class="o">.</span><span class="n">Databook</span><span class="p">([</span><span class="n">data1</span><span class="p">,</span> <span class="n">data2</span><span class="p">])</span>
<span class="k">with</span> <span class="nb">open</span><span class="p">(</span><span class="s">'book.xlsx'</span><span class="p">,</span> <span class="s">'wb'</span><span class="p">)</span> <span class="k">as</span> <span class="n">f</span><span class="p">:</span>
    <span class="n">f</span><span class="o">.</span><span class="n">write</span><span class="p">(</span><span class="n">book</span><span class="o">.</span><span class="n">xlsx</span><span class="p">)</span>
</pre></div>
</div>

file to read as first argument and the name of the Excel file to write as the
second argument.</p>

<h1>3. Modules<a class="headerlink" href="http://anandology.com/python-practice-book/modules.html#modules" title="Permalink to this headline">¶</a></h1>
<p>Modules are reusable libraries of code in Python. Python comes with many standard library modules.</p>
<p>A module is imported using the <cite>import</cite> statement.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">time</span>
<span class="gp">&gt;&gt;&gt; </span><span class="k">print</span> <span class="n">time</span><span class="o">.</span><span class="n">asctime</span><span class="p">()</span>
<span class="go">'Fri Mar 30 12:59:21 2012'</span>
</pre></div>
</div>
<p>In this example, we’ve imported the <cite>time</cite> module and called the <cite>asctime</cite>
function from that module, which returns current time as a string.</p>
<p>There is also another way to use the import statement.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">from</span> <span class="nn">time</span> <span class="kn">import</span> <span class="n">asctime</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">asctime</span><span class="p">()</span>
<span class="go">'Fri Mar 30 13:01:37 2012'</span>
</pre></div>
</div>
<p>Here were imported just the <cite>asctime</cite> function from the <cite>time</cite> module.</p>
<p>The <cite>pydoc</cite> command provides help on any module or a function.</p>
<div class="highlight-text"><div class="highlight"><pre>$ pydoc time
Help on module time:

NAME
    time - This module provides various functions to manipulate time values.
...

$ pydoc time.asctime
Help on built-in function asctime in time:

time.asctime = asctime(...)
    asctime([tuple]) -&gt; string
...
</pre></div>
</div>
<p>On Windows, the <cite>pydoc</cite> command is not available. The work-around is to use, the built-in <cite>help</cite> function.</p>
<div class="highlight-text"><div class="highlight"><pre>&gt;&gt;&gt; help('time')
Help on module time:

    time - This module provides various functions to manipulate time values.

</pre></div>

<p>Writing our own modules is very simple.</p>
<p>For example, create a file called <cite>num.py</cite> with the following content.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="k">def</span> <span class="nf">square</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>

<span class="k">def</span> <span class="nf">cube</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>
</pre></div>
</div>
<p>Now open Python interterter:</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">num</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">num</span><span class="o">.</span><span class="n">square</span><span class="p">(</span><span class="mi">3</span><span class="p">)</span>
<span class="go">9</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">num</span><span class="o">.</span><span class="n">cube</span><span class="p">(</span><span class="mi">3</span><span class="p">)</span>
<span class="go">27</span>
</pre></div>
</div>
<p>Thats all we’ve written a python library.</p>
<p>Try <cite>pydoc num</cite> (<cite>pydoc.bat numbers</cite> on Windows) to see documentation for this numbers modules. It won’t have any documentation as we haven’t providied anything yet.</p>
<p>In Python, it is possible to associate documentation for each module, function using docstrings. Docstrings are strings written at the top of the module or at the beginning of a function.</p>
<p>Lets try to document our <cite>num</cite> module by changing the contents of <cite>num.py</cite></p>
<div class="highlight-python"><div class="highlight"><pre><span class="sd">"""The num module provides utilties to work on numbers.</span>

<span class="sd">Current it provides square and cube.</span>
<span class="sd">"""</span>

<span class="k">def</span> <span class="nf">square</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="sd">"""Computes square of a number."""</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>

<span class="k">def</span> <span class="nf">cube</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="sd">"""Computes cube of a number."""</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>
</pre></div>
</div>
<p>The pydoc command will now show us the doumentation nicely formatted.</p>
<div class="highlight-text"><div class="highlight"><pre>Help on module num:

NAME
    num - The num module provides utilties to work on numbers.

FILE
    /Users/anand/num.py

DESCRIPTION
    Current it provides square and cube.

FUNCTIONS
    cube(x)
        Computes cube of a number.

    square(x)
        Computes square of a number.
</pre></div>
</div>
<p>Under the hood, python stores the documentation as a special field called <cite>__doc__</cite>.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">os</span>
<span class="gp">&gt;&gt;&gt; </span><span class="k">print</span> <span class="n">os</span><span class="o">.</span><span class="n">getcwd</span><span class="o">.</span><span class="n">__doc__</span>
<span class="go">getcwd() -&gt; path</span>

<span class="go">Return a string representing the current working directory.</span>
</pre></div>

<div class="section" id="installing-third-party-modules">
<h2>3.2. Installing third-party modules<a class="headerlink" href="http://anandology.com/python-practice-book/modules.html#installing-third-party-modules" title="Permalink to this headline">¶</a></h2>
<p>PyPI, The Python Package Index maintains the list of Python packages available. The third-party module developers usually register at PyPI and uploads their packages there.</p>
<p>The standard way to installing a python module is using <cite>pip</cite> or <cite>easy_install</cite>. Pip is more modern and perferred.</p>
<p>Lets start with installing <cite>easy_install</cite>.</p>
<ul class="simple">
<li>Download the easy_install install script <a class="reference external" href="http://peak.telecommunity.com/dist/ez_setup.py">ez_setup.py</a>.</li>
<li>Run it using Python.</li>
</ul>
<p>That will install <tt class="docutils literal"><span class="pre">easy_install</span></tt>, the script used to install third-party python packages.</p>
<p>Before installing new packages, lets understand how to manage virtual environments for installing python packages.</p>
<p>Earlier the only way of installing python packages was system wide. When used this way, packages installed for one project can conflict with other and create trouble. So people invented a way to create isolated Python environment to install packages. This tool is called <a class="reference external" href="http://www.virtualenv.org/">virtualenv</a>.</p>
<p>To install <tt class="docutils literal"><span class="pre">virtualenv</span></tt>:</p>
<blockquote>
<div>$ easy_install virtualenv</div></blockquote>
<p>Installing virtualenv also installs the <cite>pip</cite> command, a better replace for <cite>easy_install</cite>.</p>
<p>Once it is installed, create a new virtual env by running the <tt class="docutils literal"><span class="pre">virtualenv</span></tt> command.</p>
<div class="highlight-text"><div class="highlight"><pre>$ virtualenv testenv
</pre></div>
</div>
<p>Now to switch to that env.</p>
<p>On UNIX/Mac OS X:</p>
<div class="highlight-text"><div class="highlight"><pre>$ source testenv/bin/activate
</pre></div>
</div>
<p>On Windows:</p>
<div class="highlight-text"><div class="highlight"><pre>&gt; testenv\Scripts\activate
</pre></div>
</div>
<p>Now the virtualenv <cite>testenv</cite> is activated.</p>
<p>Now all the packages installed will be limited to this virtualenv. Lets try to install a third-party package.</p>
<div class="highlight-text"><div class="highlight"><pre>$ pip install tablib
</pre></div>
</div>
<p>This installs a third-party library called <tt class="docutils literal"><span class="pre">tablib</span></tt>.</p>
<p>The <tt class="docutils literal"><span class="pre">tablib</span></tt> library is a small little library to work with tabular data and write csv and Excel files.</p>

<h1>3. Modules<a class="headerlink" href="http://anandology.com/python-practice-book/modules.html#modules" title="Permalink to this headline">¶</a></h1>
<p>Modules are reusable libraries of code in Python. Python comes with many standard library modules.</p>
<p>A module is imported using the <cite>import</cite> statement.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">time</span>
<span class="gp">&gt;&gt;&gt; </span><span class="k">print</span> <span class="n">time</span><span class="o">.</span><span class="n">asctime</span><span class="p">()</span>
<span class="go">'Fri Mar 30 12:59:21 2012'</span>
</pre></div>
</div>
<p>In this example, we’ve imported the <cite>time</cite> module and called the <cite>asctime</cite>
function from that module, which returns current time as a string.</p>
<p>There is also another way to use the import statement.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">from</span> <span class="nn">time</span> <span class="kn">import</span> <span class="n">asctime</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">asctime</span><span class="p">()</span>
<span class="go">'Fri Mar 30 13:01:37 2012'</span>
</pre></div>
</div>
<p>Here were imported just the <cite>asctime</cite> function from the <cite>time</cite> module.</p>
<p>The <cite>pydoc</cite> command provides help on any module or a function.</p>
<div class="highlight-text"><div class="highlight"><pre>$ pydoc time
Help on module time:

NAME
    time - This module provides various functions to manipulate time values.
...

$ pydoc time.asctime
Help on built-in function asctime in time:

time.asctime = asctime(...)
    asctime([tuple]) -&gt; string
...
</pre></div>
</div>
<p>On Windows, the <cite>pydoc</cite> command is not available. The work-around is to use, the built-in <cite>help</cite> function.</p>
<div class="highlight-text"><div class="highlight"><pre>&gt;&gt;&gt; help('time')
Help on module time:

NAME
    time - This module provides various functions to manipulate time values.
...
</pre></div>
</div>
<p>Writing our own modules is very simple.</p>
<p>For example, create a file called <cite>num.py</cite> with the following content.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="k">def</span> <span class="nf">square</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>

<span class="k">def</span> <span class="nf">cube</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>
</pre></div>
</div>
<p>Now open Python interterter:</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">num</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">num</span><span class="o">.</span><span class="n">square</span><span class="p">(</span><span class="mi">3</span><span class="p">)</span>
<span class="go">9</span>
<span class="gp">&gt;&gt;&gt; </span><span class="n">num</span><span class="o">.</span><span class="n">cube</span><span class="p">(</span><span class="mi">3</span><span class="p">)</span>
<span class="go">27</span>
</pre></div>
</div>
<p>Thats all we’ve written a python library.</p>
<p>Try <cite>pydoc num</cite> (<cite>pydoc.bat numbers</cite> on Windows) to see documentation for this numbers modules. It won’t have any documentation as we haven’t providied anything yet.</p>
<p>In Python, it is possible to associate documentation for each module, function using docstrings. Docstrings are strings written at the top of the module or at the beginning of a function.</p>
<p>Lets try to document our <cite>num</cite> module by changing the contents of <cite>num.py</cite></p>
<div class="highlight-python"><div class="highlight"><pre><span class="sd">"""The num module provides utilties to work on numbers.</span>

<span class="sd">Current it provides square and cube.</span>
<span class="sd">"""</span>

<span class="k">def</span> <span class="nf">square</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="sd">"""Computes square of a number."""</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>

<span class="k">def</span> <span class="nf">cube</span><span class="p">(</span><span class="n">x</span><span class="p">):</span>
    <span class="sd">"""Computes cube of a number."""</span>
    <span class="k">return</span> <span class="n">x</span> <span class="o">*</span> <span class="n">x</span>
</pre></div>
</div>
<p>The pydoc command will now show us the doumentation nicely formatted.</p>
<div class="highlight-text"><div class="highlight"><pre>Help on module num:

NAME
    num - The num module provides utilties to work on numbers.

FILE
    /Users/anand/num.py

DESCRIPTION
    Current it provides square and cube.

FUNCTIONS
    cube(x)
        Computes cube of a number.

    square(x)
        Computes square of a number.
</pre></div>
</div>
<p>Under the hood, python stores the documentation as a special field called <cite>__doc__</cite>.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="gp">&gt;&gt;&gt; </span><span class="kn">import</span> <span class="nn">os</span>
<span class="gp">&gt;&gt;&gt; </span><span class="k">print</span> <span class="n">os</span><span class="o">.</span><span class="n">getcwd</span><span class="o">.</span><span class="n">__doc__</span>
<span class="go">getcwd() -&gt; path</span>

<span class="go">Return a string representing the current working directory.</span>
</pre></div>
</div>

<p>The <cite>zipfile</cite> module provides interface to read and write zip files.</p>
<p>Here are some examples to demonstate the power of zipfile module.</p>
<p>The following example prints names of all the files in a zip archive.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="kn">import</span> <span class="nn">zipfile</span>
<span class="n">z</span> <span class="o">=</span> <span class="n">zipfile</span><span class="o">.</span><span class="n">ZipFile</span><span class="p">(</span><span class="s">"a.zip"</span><span class="p">)</span>
<span class="k">for</span> <span class="n">name</span> <span class="ow">in</span> <span class="n">z</span><span class="o">.</span><span class="n">namelist</span><span class="p">():</span>
    <span class="k">print</span> <span class="n">name</span>
</pre></div>
</div>
<p>The following example prints each file in the zip archive.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="kn">import</span> <span class="nn">zipfile</span>
<span class="n">z</span> <span class="o">=</span> <span class="n">zipfile</span><span class="o">.</span><span class="n">ZipFile</span><span class="p">(</span><span class="s">"a.zip"</span><span class="p">)</span>
<span class="k">for</span> <span class="n">name</span> <span class="ow">in</span> <span class="n">z</span><span class="o">.</span><span class="n">namelist</span><span class="p">():</span>
    <span class="k">print</span>
    <span class="k">print</span> <span class="s">"FILE:"</span><span class="p">,</span> <span class="n">name</span>
    <span class="k">print</span>
    <span class="k">print</span> <span class="n">z</span><span class="o">.</span><span class="n">read</span><span class="p">(</span><span class="n">name</span><span class="p">)</span>
</pre></div>
</div>
<p><strong>Problem 11: </strong>Write a python program <cite>zip.py</cite> to create a zip file. The program should take name of zip file as first argument and files to add as rest of the arguments.</p>
<div class="highlight-text"><div class="highlight"><pre>$ python zip.py foo.zip file1.txt file2.txt
</pre></div>
</div>

<div class="section" id="installing-third-party-modules">
<h2>3.2. Installing third-party modules<a class="headerlink" href="http://anandology.com/python-practice-book/modules.html#installing-third-party-modules" title="Permalink to this headline">¶</a></h2>
<p>PyPI, The Python Package Index maintains the list of Python packages available. The third-party module developers usually register at PyPI and uploads their packages there.</p>
<p>The standard way to installing a python module is using <cite>pip</cite> or <cite>easy_install</cite>. Pip is more modern and perferred.</p>
<p>Lets start with installing <cite>easy_install</cite>.</p>
<ul class="simple">
<li>Download the easy_install install script <a class="reference external" href="http://peak.telecommunity.com/dist/ez_setup.py">ez_setup.py</a>.</li>
<li>Run it using Python.</li>
</ul>
<p>That will install <tt class="docutils literal"><span class="pre">easy_install</span></tt>, the script used to install third-party python packages.</p>
<p>Before installing new packages, lets understand how to manage virtual environments for installing python packages.</p>
<p>Earlier the only way of installing python packages was system wide. When used this way, packages installed for one project can conflict with other and create trouble. So people invented a way to create isolated Python environment to install packages. This tool is called <a class="reference external" href="http://www.virtualenv.org/">virtualenv</a>.</p>
<p>To install <tt class="docutils literal"><span class="pre">virtualenv</span></tt>:</p>
<blockquote>
<div>$ easy_install virtualenv</div></blockquote>
<p>Installing virtualenv also installs the <cite>pip</cite> command, a better replace for <cite>easy_install</cite>.</p>
<p>Once it is installed, create a new virtual env by running the <tt class="docutils literal"><span class="pre">virtualenv</span></tt> command.</p>
<div class="highlight-text"><div class="highlight"><pre>$ virtualenv testenv
</pre></div>
</div>
<p>Now to switch to that env.</p>
<p>On UNIX/Mac OS X:</p>
<div class="highlight-text"><div class="highlight"><pre>$ source testenv/bin/activate
</pre></div>
</div>
<p>On Windows:</p>
<div class="highlight-text"><div class="highlight"><pre>&gt; testenv\Scripts\activate
</pre></div>
</div>
<p>Now the virtualenv <cite>testenv</cite> is activated.</p>
<p>Now all the packages installed will be limited to this virtualenv. Lets try to install a third-party package.</p>
<div class="highlight-text"><div class="highlight"><pre>$ pip install tablib
</pre></div>
</div>
<p>This installs a third-party library called <tt class="docutils literal"><span class="pre">tablib</span></tt>.</p>
<p>The <tt class="docutils literal"><span class="pre">tablib</span></tt> library is a small little library to work with tabular data and write csv and Excel files.</p>
<p>Here is a simple example.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="c"># create a dataset</span>
<span class="n">data</span> <span class="o">=</span> <span class="n">tablib</span><span class="o">.</span><span class="n">Dataset</span><span class="p">()</span>

<span class="c"># Add rows</span>
<span class="n">data</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"A"</span><span class="p">,</span> <span class="mi">1</span><span class="p">])</span>
<span class="n">data</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"B"</span><span class="p">,</span> <span class="mi">2</span><span class="p">])</span>
<span class="n">data</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"C"</span><span class="p">,</span> <span class="mi">3</span><span class="p">])</span>

<span class="c"># save as csv</span>
<span class="k">with</span> <span class="nb">open</span><span class="p">(</span><span class="s">'test.csv'</span><span class="p">,</span> <span class="s">'wb'</span><span class="p">)</span> <span class="k">as</span> <span class="n">f</span><span class="p">:</span>
    <span class="n">f</span><span class="o">.</span><span class="n">write</span><span class="p">(</span><span class="n">data</span><span class="o">.</span><span class="n">csv</span><span class="p">)</span>

<span class="c"># save as Excel</span>
<span class="k">with</span> <span class="nb">open</span><span class="p">(</span><span class="s">'test.xls'</span><span class="p">,</span> <span class="s">'wb'</span><span class="p">)</span> <span class="k">as</span> <span class="n">f</span><span class="p">:</span>
    <span class="n">f</span><span class="o">.</span><span class="n">write</span><span class="p">(</span><span class="n">data</span><span class="o">.</span><span class="n">xls</span><span class="p">)</span>

<span class="c"># save as Excel 07+</span>
<span class="k">with</span> <span class="nb">open</span><span class="p">(</span><span class="s">'test.xlsx'</span><span class="p">,</span> <span class="s">'wb'</span><span class="p">)</span> <span class="k">as</span> <span class="n">f</span><span class="p">:</span>
    <span class="n">f</span><span class="o">.</span><span class="n">write</span><span class="p">(</span><span class="n">data</span><span class="o">.</span><span class="n">xlsx</span><span class="p">)</span>
</pre></div>
</div>
<p>It is even possible to create multi-sheet excel files.</p>
<div class="highlight-python"><div class="highlight"><pre><span class="n">sheet1</span> <span class="o">=</span> <span class="n">tablib</span><span class="o">.</span><span class="n">Dataset</span><span class="p">()</span>
<span class="n">sheet1</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"A1"</span><span class="p">,</span> <span class="mi">1</span><span class="p">])</span>
<span class="n">sheet1</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"A2"</span><span class="p">,</span> <span class="mi">2</span><span class="p">])</span>

<span class="n">sheet2</span> <span class="o">=</span> <span class="n">tablib</span><span class="o">.</span><span class="n">Dataset</span><span class="p">()</span>
<span class="n">sheet2</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"B1"</span><span class="p">,</span> <span class="mi">1</span><span class="p">])</span>
<span class="n">sheet2</span><span class="o">.</span><span class="n">append</span><span class="p">([</span><span class="s">"B2"</span><span class="p">,</span> <span class="mi">2</span><span class="p">])</span>


<span class="n">book</span> <span class="o">=</span> <span class="n">tablib</span><span class="o">.</span><span class="n">Databook</span><span class="p">([</span><span class="n">data1</span><span class="p">,</span> <span class="n">data2</span><span class="p">])</span>
<span class="k">with</span> <span class="nb">open</span><span class="p">(</span><span class="s">'book.xlsx'</span><span class="p">,</span> <span class="s">'wb'</span><span class="p">)</span> <span class="k">as</span> <span class="n">f</span><span class="p">:</span>
    <span class="n">f</span><span class="o">.</span><span class="n">write</span><span class="p">(</span><span class="n">book</span><span class="o">.</span><span class="n">xlsx</span><span class="p">)</span>
</pre></div>
</div>
